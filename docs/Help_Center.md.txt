# 🆘 Help Center – EngageTrack AI

Welcome to **EngageTrack AI** – a simulated SaaS analytics platform for demonstrating AI-powered insights, churn risk prediction, and feature nudging.

---

## 👤 1. Select a User

Use the **dropdown menu** to choose a simulated user. Their full profile — including persona, engagement level, and churn risk — will be displayed.

---

## 💡 2. Generate AI-Driven Recommendations

- Click 🔁 **Generate New Nudge** to simulate a fresh engagement tip
- AI nudges are tailored by persona and recent behavior
- Feature suggestions and lifecycle tips are shown per user

---

## 📥 3. Download User Summary

Click the **Download Summary** button to export all user insights (persona, plan, churn score, nudge) as a TXT file — useful for reporting or auditing.

---

## 📊 4. Explore the System Dashboard

Navigate to the **Analytics Dashboard tab** to view:

- 🔥 Engagement Level Breakdown
- 🚨 Churn Risk Distribution
- 💡 Plan Type Segmentation
- 🧪 A/B Variant Distribution

These charts reflect the system-wide simulated user base.

---

## 👩‍💻 For Developers

| Task                     | File/Module                |
|--------------------------|----------------------------|
| Modify user data         | `data/churn.csv`           |
| Tweak nudge responses    | `src/mock_api.py`          |
| Update ML model behavior | `src/app.py`               |
| Extend analytics         | `src/app.py` (tab 2 logic) |

All source code is modular under the `src/` directory. Streamlit runs the app from `src/app.py`.

# 📦 Release Notes – EngageTrack AI

---

## 🚀 Version 1.0 – Initial Public Release  
📅 **Date:** May 2025  
🔧 **Maintainer:** [Tanesh Singhal](https://github.com/taneshsin)

---

### ✅ Features Implemented

| Category                 | Description |
|--------------------------|-------------|
| 🧠 AI Nudges              | Simulated GPT-style suggestions personalized by user persona |
| 📉 Churn Risk Prediction  | Real XGBoost model using behavioral data |
| 📊 Visual Dashboard       | Engagement and churn distribution across user base |
| 🧪 A/B Variant Testing     | Users randomly assigned to Variant A or B |
| 👤 Per-User Detail View    | Includes persona, plan, churn risk, and recommendations |
| 📥 Summary Download       | One-click export of user lifecycle summary (TXT) |
| 🧾 Usage Logging           | Activity logged securely to `/tmp/usage.log` |
| 🧱 Containerization        | Dockerfile with non-root user for production safety |
| 📁 Modular Architecture   | Organized Python codebase (`src/`) for scalability |
| 🌐 Streamlit UI           | Clean, tabbed layout powered by Streamlit |

---

### 🔧 DevOps & Deployment

- CI/CD via **GitHub Actions**  
- Deployed on **Azure Kubernetes Service (AKS)**  
- NGINX Ingress configured with rate limiting  
- Secure secrets management via GitHub repo settings  
- `.dockerignore`, `.gitignore`, and non-root Docker best practices enforced
